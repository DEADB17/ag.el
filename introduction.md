# A Silver Searcher frontend for Emacs

The Silver Search is a *really* fast text search tool. ag.el provides
an Emacsy frontend for your searches.

[ag.el has thorough documentation](https://wilfredh.gitbooks.io/ag-el/content/). If
you're a new user, start with
[Installation](https://wilfredh.gitbooks.io/ag-el/content/docs/installation.html)
then
[Your First Search](https://wilfredh.gitbooks.io/ag-el/content/docs/your_first_search.html).
